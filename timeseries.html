Time Series dataset for prediction#

Pengambilan Data NO‚ÇÇ Sentinel-5P untuk Wilayah Bangkalan melalui openEO#

Kode berikut digunakan untuk mengambil data konsentrasi Nitrogen Dioksida (NO‚ÇÇ) dari satelit Sentinel-5P melalui layanan openEO. Data ini digunakan untuk analisis kualitas udara berdasarkan waktu dan wilayah tertentu.

Tahapan proses:

  1. Autentikasi ke openEO
    Dilakukan untuk mendapatkan akses ke dataset Sentinel-5P melalui endpoint openeo.dataspace.copernicus.eu menggunakan login OIDC.

  2. Menentukan Area of Interest (AOI)
    AOI didefinisikan dalam bentuk polygon (koordinat lintang dan bujur) yang mewakili area kecil di Bangkalan yang menjadi lokasi penelitian.

  3. Memuat koleksi data Sentinel-5P
    Koleksi SENTINEL_5P_L2 dimuat dengan rentang waktu tertentu (tanggal mulai sampai tanggal akhir) dan bounding box lebih luas dari AOI untuk menghindari error resolusi spasial karena Sentinel-5P memiliki resolusi ~7 km.

  4. Agregasi Temporal (rata-rata harian)
    Karena data Sentinel-5P tersedia beberapa kali dalam satu hari, maka dilakukan perhitungan rata-rata per hari untuk mendapatkan nilai harian.

  5. Agregasi Spasial berdasarkan AOI
    Rata-rata nilai NO‚ÇÇ dihitung hanya untuk area yang telah ditentukan (Bangkalan).

  6. Eksekusi batch dan penyimpanan
    Proses dikirim ke server dalam bentuk batch job, dan hasilnya disimpan dalam format .nc (NetCDF) untuk analisis lebih lanjut.

        "west": 112.60,
        "south": -7.10,

# 4. Hitung rata-rata harian

    period="day",
    reducer="mean"

    geometries=aoi,
    reducer="mean"
)

Membaca Hasil Ekspor NetCDF dan Mengekstrak Variabel NO‚ÇÇ#

Setelah proses pengambilan data dari openEO berhasil dan hasilnya disimpan dalam format NO2_in_bangkalan.nc, langkah selanjutnya adalah membaca isi file tersebut dan melihat variabel apa saja yang tersedia.

Format NetCDF (.nc) adalah format standar untuk data iklim dan satelit yang mendukung penyimpanan multi-dimensi (misalnya data spasial dan temporal). Kita akan menggunakan pustaka netCDF4 untuk membuka dan membaca file ini.

Langkah-langkah utama:

  1. Memuat file .nc menggunakan netCDF4.Dataset File dataset dibuka dan siap diakses.

  2. Menampilkan daftar variabel yang tersedia Ini penting untuk mengetahui nama variabel NO‚ÇÇ dan variabel waktu yang digunakan (misalnya "time" atau "t").

  3. Mengambil data NO‚ÇÇ dan waktu Data NO‚ÇÇ dibaca sebagai array multi-dimensi. Waktu biasanya dalam format bilangan offset (mis. ‚Äúdays since 1970-01-01‚Äù).

  4. Konversi waktu ke format tanggal Jika metadata waktu memiliki atribut units, kita dapat mengubahnya ke format datetime.

  5. Menampilkan contoh data Untuk melihat apakah pembacaan data berhasil, kita tampilkan beberapa baris pertama.

import netCDF4

file_path = "/workspaces/PSD1/Tugas/data/NO2_in_bangkalan.nc"
ds = netCDF4.Dataset(file_path)

# Lihat seluruh variabel yang tersedia
print("üì¶ Variabel dalam file:")
print(ds.variables.keys())

# Misal variabel utama bernama "NO2"
no2 = ds.variables["NO2"][:]

# Waktu (biasanya dalam format 'days since ...' atau sejenis)
time = ds.variables["t"][:]

# Konversi waktu ke format tanggal jika punya atribut 'units'
try:
    time_units = ds.variables["t"].units
    dates = netCDF4.num2date(time, units=time_units)
except Exception:
    dates = time  # fallback kalau tidak ada units

# Tampilkan 10 baris pertama
print("\nüìÖ Contoh data pertama:")
for i in range(min(10, len(no2))):
    print(f"{dates[i]} | NO2: {no2[i]}")

Konversi Data NetCDF ke CSV untuk Analisis Lebih Lanjut#

Hasil data dari openEO berupa file NetCDF (.nc) yang merupakan format standar untuk data penginderaan jauh dan data sains bumi. Namun, untuk keperluan analisis statistik dan visualisasi sederhana (misalnya menggunakan Excel atau Pandas), data lebih mudah digunakan jika dikonversi ke format CSV.

import netCDF4
import pandas as pd
import numpy as np

# 1. Baca file NetCDF
file_path = "costs/NO2_in_bangkalan.nc"
ds = netCDF4.Dataset(file_path)

# 2. Tampilkan struktur file
print(ds)

# 3. Ambil variabel yang tersedia
print(ds.variables.keys())

# 4. Ambil variabel waktu dan NO2
time_var = ds.variables["t"][:]
no2_var = ds.variables["NO2"][:]

# 5. Konversi waktu ke datetime
time_units = ds.variables["t"].units
dates = netCDF4.num2date(time_var, units=time_units)

# 6. Jika data NO2 berdimensi lebih dari 1 (misalnya [time, lat, lon]),
# kita rata-ratakan agar menjadi satu nilai per waktu
if no2_var.ndim > 1:
    no2_mean = np.nanmean(no2_var, axis=tuple(range(1, no2_var.ndim)))
else:
    no2_mean = no2_var

# 7. Buat DataFrame
df = pd.DataFrame({
    "time": dates,
    "NO2": no2_mean
})

# 8. Simpan ke CSV
df.to_csv("costs/NO2_bangkalan.csv", index=False)
print("‚úÖ File NO2_bangkalan.csv berhasil dibuat.")
print(df.head())

Normalisasi Data untuk Model Supervised Learning#

Setelah data deret waktu NO‚ÇÇ berhasil diubah ke bentuk supervised (memiliki variabel lag), langkah berikutnya adalah melakukan normalisasi menggunakan teknik Min-Max Scaling. Normalisasi ini bertujuan untuk menyeragamkan skala nilai fitur sehingga seluruh input model berada dalam rentang [0, 1].

Pemodelan time series modern seperti LSTM dan GRU sangat sensitif terhadap perbedaan skala data, sehingga normalisasi menjadi proses penting untuk meningkatkan stabilitas dan akurasi model.

from sklearn.preprocessing import MinMaxScaler

# Ubah index time jadi kolom lagi

# Konversi kembali ke Dataframe

Pemodelan Time Series NO‚ÇÇ Menggunakan K-Nearest Neighbors (KNN) Regressor#

Setelah data NO‚ÇÇ dinormalisasi dan diubah ke bentuk supervised learning, langkah berikutnya adalah membangun model machine learning untuk melakukan peramalan. Pada bagian ini digunakan K-Nearest Neighbors Regressor (KNN) sebagai model dasar (baseline model) sebelum beralih ke model berbasis deep learning seperti LSTM atau GRU.

import numpy as np
import matplotlib.pyplot as plt
from sklearn import neighbors
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error, r2_score

X = supervised[['NO2(t-4)', 'NO2(t-3)', 'NO2(t-2)', 'NO2(t-1)']]
y = supervised['NO2(t)']

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, shuffle=False)

n_neighbors = 5

plt.figure(figsize=(10, 6))

for i, weights in enumerate(['uniform', 'distance']):
    knn = neighbors.KNeighborsRegressor(n_neighbors, weights=weights)
    knn.fit(X_train, y_train)
    y_pred = knn.predict(X_test)

    rmse = np.sqrt(mean_squared_error(y_test, y_pred))
    r2 = r2_score(y_test, y_pred)

    plt.subplot(2, 1, i + 1)
    plt.plot(range(len(y_test)), y_test, 'darkorange', label="Actual")
    plt.plot(range(len(y_pred)), y_pred, 'navy', label="Prediction")
    plt.title(f"KNeighborsRegressor (K={n_neighbors}, weight='{weights}'\nRMSE={rmse:.6f}, R2={r2:.4f})")
    plt.legend()
    plt.tight_layout()

plt.show()
